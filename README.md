# 🎄 Christmas Tree Animation 🎄

A simple, festive Christmas tree built with HTML, CSS, and JavaScript.

##  Live Demo
Open `index.html` in any modern browser to see the magic!

##  Features
- Beautifully decorated Christmas tree
- Twinkling star on top
- Falling snow animation
- Colorful ornaments that change colors
- (Add your own features here)

## Tech Stack
- HTML5 for structure
- CSS3 for styling & animations
- JavaScript for interactivity & effects

##  How to Run
1. Clone/download this project
2. Double-click `index.html`
3. Enjoy the Christmas spirit! 

## Customize
- Change tree colors in `style.css`
- Modify ornament speed in `script.js`
- Adjust snow density in JavaScript

##  Project Structure
```
/
├── index.html      # Main structure
├── style.css       # Styling & animations
└── script.js       # Tree logic & effects
```
